Ключове слово Static
1. В статичному методі можна викликати тільки статичні методи.
2. В статичому методі доступні тільки статичні змінні, обявлені в його класі.
3. Статичний метод неможе посилатися на this поля.
4. Статични метод або статичні поля і методи доступні не по силці на обєкт,
   а по імені класу, статична змінна загальна для всіх екземплярів класу.

Static блоки
1. Використовуються для ініціалізації статичних змінних без створення обєкта цього класу.
2. Статичні блоки виконуються при першій загрузці класу.
